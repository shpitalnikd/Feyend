# Feyend

A location-based item-finding marketplace. Post a request to find an item at a store near you, offer a reward, and let the community find it for you.

## Built With

- [Expo](https://expo.dev) (SDK 54) + Expo Router
- [Supabase](https://supabase.com) (Postgres + PostGIS, Auth, Storage)
- react-native-maps

## Getting Started

1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env` and add your Supabase credentials
4. `npm start` — scan QR with Expo Go
